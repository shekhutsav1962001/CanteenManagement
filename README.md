# How_To_Run

- ## How To run backend 

```
$ git clone https://github.com/shekhutsav1962001/CanteenManagement
$ cd CanteenManagement

$ cd backend
$ npm install

$ npm run dev
```

- ## How To run Front-end App(Angular)

```
$ git clone https://github.com/shekhutsav1962001/CanteenManagement
$ cd CanteenManagement

$ cd frontend
$ cd canteen
$ npm install

$ ng serve 
```
